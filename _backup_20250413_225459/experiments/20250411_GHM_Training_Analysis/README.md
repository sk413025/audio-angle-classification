## 實驗摘要

本實驗分析了使用 Gradient Harmonizing Mechanism (GHM) 進行模型訓練的效果，特別關注其在處理不平衡數據集時的表現。GHM 通過自動重新加權樣本來優化訓練過程，尤其適合處理樣本分布不均勻的情況。

實驗結果表明 GHM 能夠：
1. 自動平衡不同難度樣本的梯度貢獻
2. 提供更穩定的訓練過程
3. 在處理不平衡數據集時表現優異
